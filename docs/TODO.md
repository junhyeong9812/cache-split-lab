## 다음 작업 — 3번: 레디스 외부화 (arm D, 미착수)

**별도 정의 게이트부터** (신규 코드: compose + Lua — 재트리아지 필요).

- **arm D — 공유 레디스 + 라운드로빈.** "캐시를 외부로 빼면?"
- **주의: Redis 기본 LRU 는 우리 것과 다르다** (바이트 단위 + 샘플링 근사)
  → **Lua 로 ZSET+HASH LRU 직접 구현**해야 엔트리 수 정확·이론 유지.
- **히트율은 잴 필요 없음** — 캐시 하나(10,000)라 A 와 동일(연역). 잴 것은
  **지연·포화점** = 외부화의 대가(레디스 싱글스레드 직렬화 + 매 요청 왕복).
- **마스터/레플리카(arm E)는 사용자와 재논의**: 읽기를 레플리카로 보내면 마스터 LRU 가
  읽기를 못 봐 FIFO 로 퇴화 — "왜 안 되나"를 보는 실험.
- 개념 대응: A=스케일업 / B=나이브 스케일아웃 / C=분할 스케일아웃 / D=외부화 스케일업.
