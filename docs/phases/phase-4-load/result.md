## 특기 사항 (해석은 gate.md)

1. **C 의 median 붕괴가 5000 부터 시작**: p50 이 0.36 → 16 → 39 → 95ms 로 계단마다 배가.
   같은 지점 B 의 p50 은 5500 까지 0.37ms 를 유지. 반대로 5000 에서 **p95 는 B(112)가
   C(109)보다 근소하게 나쁘다** — B 의 꼬리는 미스(5ms origin 왕복+대기), C 의 몸통은
   핫 노드 스로틀링. **두 arm 은 다른 축으로 아프다.**
2. **C 는 60s 창에서 6000 을 드롭 없이 달성** (Phase 0 30s 창에선 드롭 1.84%) — 도착률은
   소화하지만 p50 95ms. 드롭은 창 길이·VU 여유에 민감한 취약한 기준이고, 지연이 견고한
   신호다.
3. **첫 지점(4000)의 히트율이 전 arm 에서 눌려 있다** (A 0.9913 → 이후 0.999x): 워밍업이
   SEED=1 ·2,000 RPS·60s 라 측정(SEED=42)의 꼬리 키를 다 못 데웠다 — 첫 지점이 남은
   수렴을 흡수. 다음 스윕부턴 워밍업을 측정 시드로/더 길게.
